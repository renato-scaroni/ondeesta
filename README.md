Onde está?
======

-> Descrição:
Um projeto pessoal escrito por Renato Scaroni.
Conjunto de scripts em python que rastreia encomendas no correio brasileiro,
com suporte a multi-usuarios e multi-encomendas e notificação de alteração de
status via e-mail.
